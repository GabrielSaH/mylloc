## Resumo do projeto
Bem-vindo ao projeto de Gerenciador de Memória para nosso Sistema Operacional Hipotético! Este é um projeto open-source voltado para o desenvolvimento de um gerenciador de memória simples porem robusto, sendo efetuado para a disciplina de Sistemas operacionais da faculdade Instituto Federal Catarinense. O projeto sera desenvolvido em C tendo em mente um sistema operacional para os proccessadores da linha 80/86, apenas sera desenvolvido um GERENCIADOR de memoria e não um ALOCADOR de memoria.
